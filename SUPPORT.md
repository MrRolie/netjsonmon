# Support

## Where to Ask for Help

- Usage questions: open a GitHub issue with context and command examples.
- Bug reports: use the bug report issue template.
- Feature requests: use the feature request template.

## What to Include

- OS and Node.js version
- `netjsonmon` version
- Command run
- Error output
- Minimal steps to reproduce

## Security Issues

For security-sensitive reports, do not open a public issue.
Use private GitHub vulnerability reporting as described in [SECURITY.md](SECURITY.md).
