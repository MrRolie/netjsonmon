### Added

- Initial public release of `netjsonmon`.
- CLI capture workflow (`run`, `inspect`, `endpoints`, `label`, `train`, `init`, `mcp`).
- Session persistence (`--useSession`, `--saveSession`), watch mode, stealth, proxy support, and MCP server integration.
